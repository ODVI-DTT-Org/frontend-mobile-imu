```
Field agent app for client visits, touchpoints & itinerary management with offline sync
```
- **Maximum:** 80 characters
- **Purpose:** Quick summary in search results
- **Tips:** Include key features and target users

#### Full Description

**Copy this to Play Console:**
```
IMU (Itinerary Manager Uniformed) is a comprehensive mobile application designed for field agents managing client visits and touchpoints for retired police personnel.

KEY FEATURES:

• Client Management - View and manage client information offline with instant search and filtering capabilities

• Touchpoint Tracking - Record visit and call touchpoints with GPS location verification, photo capture, and audio notes

• Itinerary Management - Daily schedules with route optimization and client visit tracking

• Offline-First Design - Works without internet connection, automatically syncs when online

• GPS Location - Automatic location capture for touchpoint verification and address validation

• Secure Authentication - PIN-based authentication with biometric support (fingerprint/Face ID)

• Real-Time Sync - Automatic data synchronization when internet connection is available

• Role-Based Access - Tailored experience for field agents (Caravan), telemarketers (Tele), and managers

• Photo & Audio Notes - Capture photos and voice recordings for detailed touchpoint documentation

TARGET USERS:

• Field agents (Caravan) managing in-person client visits
• Telemarketers (Tele) making follow-up calls
• Area managers overseeing field operations and team performance

SECURITY & PRIVACY:

• All data encrypted during transmission and storage
• PIN-based authentication with optional biometric login
• Session timeout after 15 minutes of inactivity
• Role-based access control for data protection

PERMISSIONS EXPLAINED:

• Location - For GPS tracking and address verification during touchpoints
• Camera - For capturing photos during client visits
• Microphone - For recording voice notes during touchpoints
• Storage - For offline data storage and document access
• Network - For API communication and data synchronization

OFFLINE CAPABILITY:

The app is designed to work without an internet connection. All client data and itineraries are stored locally on your device. When you're back online, the app automatically syncs your changes with the server.

DASHBOARD FEATURES:

• Quick access to all major features from home screen
• Real-time sync status indicator
• Today's tasks and upcoming visits overview
• Client search with advanced filtering

Perfect for field teams managing client relationships with offline reliability and secure data management.
```
- **Maximum:** 4000 characters
- **Format:** Plain text (no HTML)
- **Language:** English
